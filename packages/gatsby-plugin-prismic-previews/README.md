Integrate live [Prismic Previews][prismic-previews] into a static [Gatsby][gatsby] site to enable editors a seamless content editing experience.

- ü§ù &nbsp;Integrates tightly with the [Gatsby Prismic source plugin][gatsby-source-prismic]
- üëÅ &nbsp;Refreshes preview content automatically as changes are saved in Prismic
- üöÖ &nbsp;Adds the [Prismic Toolbar][prismic-toolbar] with an in-app edit button and preview link sharing.
- üå© &nbsp;No extra infrastructure or costs required (specifically, [Gatsby Cloud][gatsby-cloud] is not required)

## Install

```bash
npm install gatsby-plugin-prismic-previews gatsby-source-prismic gatsby-plugin-image
```
